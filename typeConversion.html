Type Conversion and Truthy and Falsey

Rules of Type Conversion

When using == to compare

  • If two values have the same type, just compare them.
  • If two values have different types, try to convert them into the same type, and then compare them.

How types are converted for comparison

  1. Number and a string: try to convert the string to a number.
  2. Boolean with any other type: convert the boolean to a number. true -> 1, false -> 0.
  3. null and undefined: true

Truthy and Falsey

The five falsey values (everything else is truthy):

  • undefined
  • null
  • 0
  • ""
  • NaN
