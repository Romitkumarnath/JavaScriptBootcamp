Variables

Naming Rules

  • JavaScript is case-sensitive. This means myVariable, MyVariable, and MYVARIABLE are three different variables.
  • Start with Letters, $, or _: Variable names must begin with a letter (a-z, A-Z), an underscore (_), or a dollar sign ($).
  • After the first character, variable names can include letters, digits (0-9), dollar signs ($), and underscores (_).
  • Variable names cannot use JavaScript reserved words (like var, let, const, if, else, for, etc.).
